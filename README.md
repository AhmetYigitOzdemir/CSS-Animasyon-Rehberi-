# ✨ CSS Paragraf Animasyonları Kütüphanesi

Modern web projeleri için hazırlanmış, estetik ve kolay kullanılabilir CSS paragraf animasyonları koleksiyonu.

## 🚀 Özellikler

- Modern ve responsive tasarım
- Glassmorphism kart yapısı
- Canlı animasyon önizlemeleri
- Tek tıkla kod kopyalama
- Animasyonları yeniden oynatma özelliği
- Tamamen HTML, CSS ve JavaScript ile geliştirilmiştir

## 🎨 İçerdiği Animasyonlar

### 1. Fade Up
Metnin aşağıdan yukarıya doğru yumuşak bir şekilde görünmesini sağlar.

### 2. Typing Effect
Daktilo yazısı efektiyle metni harf harf gösterir.

### 3. Gradient Flow
Metin üzerinde hareketli renk geçişleri oluşturur.

### 4. Glow Pulse
Neon benzeri ışık ve parıltı efekti sağlar.

### 5. Floating Effect
Metni havada süzülüyormuş gibi hareket ettirir.

### 6. Blur Reveal
Metni bulanık başlayıp netleşecek şekilde gösterir.

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts (Inter & Outfit)

## 📷 Önizleme

Proje içerisinde her animasyon için:
- Canlı demo
- Kaynak kod görüntüleme
- Kopyalama butonu
- Yeniden oynatma butonu

bulunmaktadır.
