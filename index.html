✨ Paragraf Animasyonları

Projelerinize hayat katacak modern ve estetik CSS animasyon örnekleri

Aşağıdan Süzülme (Fade Up)

Bu paragraf, sayfa yüklendiğinde aşağıdan yukarıya doğru süzülerek ve şeffaflığı azalarak yumuşakça belirir. 

/* HTML */

Bu bir animasyonlu paragraftır.


/* CSS */
.fade-up {
  opacity: 0;
  animation: fadeUp 1.2s cubic-bezier(0.25, 1, 0.5, 1) forwards;
}

@keyframes fadeUp {
  from {
    opacity: 0;
    transform: translateY(25px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
Daktilo Efekti (Typing)

Yazı daktiloda yazılıyormuş gibi harf harf belirir. 

/* HTML */

Harf harf yazılan metin.


/* CSS */
.typing {
  display: inline-block;
  overflow: hidden;
  border-right: 3px solid #ec4899;
  white-space: nowrap;
  width: 0;
  animation: 
    typing 3s steps(30, end) forwards,
    blink-caret 0.75s step-end infinite;
}

@keyframes typing {
  from { width: 0 }
  to { width: 100% }
}

@keyframes blink-caret {
  from, to { border-color: transparent }
  50% { border-color: #ec4899 }
}
Renk Geçişi (Gradient Flow)

Canlı ve dinamik gradyan renkleri metin boyunca sürekli dalgalanır. 

/* HTML */

Renkli dalgalı paragraf.


/* CSS */
.gradient-text {
  background: linear-gradient(120deg, #ff007f, #7f00ff, #00f0ff, #ff007f);
  background-size: 300% 300%;
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  animation: gradientMove 4s ease infinite;
  font-weight: 600;
}

@keyframes gradientMove {
  0% { background-position: 0% 50% }
  50% { background-position: 100% 50% }
  100% { background-position: 0% 50% }
}
Işıma Efekti (Glow Pulse)

Metnin arkasında neon benzeri parıldayan, yumuşak bir ışık efekti. 

/* HTML */

Parıldayan neon metin.


/* CSS */
.glow-text {
  animation: glowPulse 2s ease-in-out infinite alternate;
}

@keyframes glowPulse {
  from {
    text-shadow: 0 0 4px rgba(255,255,255,0.2), 
                 0 0 10px rgba(99,102,241,0.2);
  }
  to {
    text-shadow: 0 0 8px rgba(255,255,255,0.6), 
                 0 0 20px rgba(99,102,241,0.8),
                 0 0 30px rgba(236,72,153,0.6);
  }
}
Uçuşan Efekt (Float)

Havada asılı kalmış gibi sürekli aşağı ve yukarı hafifçe yüzen bir paragraf. 

/* HTML */

Uçan ve sallanan paragraf.


/* CSS */
.float-text {
  animation: floatUpDown 3s ease-in-out infinite;
}

@keyframes floatUpDown {
  0% { transform: translateY(0px); }
  50% { transform: translateY(-10px); }
  100% { transform: translateY(0px); }
}
Bulanıklıktan Netleşme

İlk başta bulanık ve okunamaz olan metin, pürüzsüzce odaklanarak netleşir. 

/* HTML */

Bulanıklıktan odağa giren metin.


/* CSS */
.blur-reveal {
  opacity: 0;
  filter: blur(10px);
  animation: blurReveal 1.5s cubic-bezier(0.25, 1, 0.5, 1) forwards;
}

@keyframes blurReveal {
  from {
    opacity: 0;
    filter: blur(10px);
    transform: scale(0.95);
  }
  to {
    opacity: 1;
    filter: blur(0);
    transform: scale(1);
  }
}
